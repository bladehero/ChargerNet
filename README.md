### Stack of technologies:
- .NET Standard 2.0
- Xamarin.Forms
- SQLite (SQL)
- QRCoder (https://github.com/codebude/QRCoder)

#### Main Principles used of SOLID and other architecture patterns:
- Interface Segregation
- Dependency Inversion
- MVVM
- Singleton
